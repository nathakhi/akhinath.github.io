Project Overview

This project focused on predicting customer churn for a credit card service using machine learning techniques. A dataset of over 10,000 customers from Kaggle was analyzed, exploring various customer attributes such as income, gender, and card usage patterns. After cleaning and preprocessing the data, multiple models including Logistic Regression, Decision Trees, and Random Forests were applied.

Problem Statement

A bank manager was concerned with the increasing number of customers leaving their credit card services. The project aimed to predict which customers are prone to churn, based on customer and credit card consumption attributes, and reduce the churning proportion to less than 10% over the next six months.

Data Description

  • Source: Kaggle dataset
  • Size: 10,127 customers
  • Features: 21 columns, main features are Attrition Flag, Gender, Educational Level, Marital Status, Income Category, Card Category, Total Revolving Balance, Average Open to Buy, etc.

Methods:

Data Preprocessing:

  • Imported necessary libraries
  • Loaded and inspected the data
  • Handled missing values and duplicate entries
  • Removed irrelevant "Unknown" categories
  • Converted categorical variables to numerical values
  • Exported cleaned data with 7,081 rows and 21 columns

Exploratory Data Analysis (EDA)

  • Performed descriptive statistics on key variables
  • Visualized attrition by gender, income, marital status, and card category
  • Analyzed correlations between features like credit limit, utilization ratio, and attrition

Data Visualization

  • Using Tableau visualizing churn trends.
  • Plotting feature importance to understand which variables contribute the most to churn.

See the dashboard in tableau public!

Hypothesis Testing

  • H0 = There is no difference in attrition between men and women
  • HA = There is difference in attrition between men and women
  • Test: Chi-square Test of Homogeneity
  • Chi-Square Statistic: 8.8523959
  • P-value: 0.003 (<0.05)
  • Conclusion: Reject the Null hypothesis

Machine Learning Models

  • Data split into training and testing sets (80/20)
  • Features selected using Mutual Information Gain
  • Feature scaling (Standardization) applied
  • Trained multiple models: Logistic Regression, Decision Tree, Random Forest, K-Nearest Neighbors, Artificial Neural Network

Model Performance Comparison

  • Use metrics like Accuracy, Precision, Recall, and F1-Score to evaluate model performance.
  • The Random Forest model performed best with an accuracy of 96%, precision of 0.75, recall of 0.79, and F1-score of 0.77.

Summary of Findings

  • The attrition rate was higher among customers in the extreme income categories (< $40k: 16.94% and > $120k: 17.83%).
  • Women had a 2.58% higher attrition rate compared to men.
  • Platinum card holders had the highest attrition rate at 27.27%.
  • Random Forest Classifier can better classify the customers who are going to get churned (Accuracy 96%).

Limitations

  • Imbalanced dataset (15.72% attrited customers).
  • No hyperparameter tuning was applied to models.
  • Weak correlation between most features and the target variable.
